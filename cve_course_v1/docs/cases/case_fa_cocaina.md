# Caso — FA por cocaína (instável)

- **Monitor**: RR irregular, FC ~160–180, sem onda P visível.
- **Clínica**: dor 7–8/10, sudorese, náusea.
- **Conduta**: **CVE sincronizada 200 J** → RS; entre tentativas **benzo + Mg 2 g IV**; sem β‑bloqueador puro.
- **Pós**: ECG documentado, orientação de **abstinência** e sinais de alarme.
