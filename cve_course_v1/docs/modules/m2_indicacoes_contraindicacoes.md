# Módulo 2 — Indicações & Contraindicações

## 1. Indicação clássica
- Taquiarritmia **com pulso** + **instabilidade** clínica/hemodinâmica → **CVE imediata**.

## 2. Contraindicações relativas
- **FA estável** de longa data sem anticoagulação adequada (risco de êmbolo) — avaliar balanceado; instabilidade sempre prevalece.
- **Digoxina**: risco de arritmias pós-CVE se toxicidade (corrigir antes se possível).
- **Recursos limitados**: priorize segurança mínima (monitor, O₂ sob demanda, sedação curta).

## 3. Sedoanalgesia
- **Etomidato 0,15 mg/kg IV** (estável hemodinamicamente).
- Analgesia: **fentanil 0,5–1 mcg/kg IV** (se disponível) ou benzo para **blunting simpático**.
