# Módulo 1 — Mecanismos & Instabilidade

## 1. Mecanismos eletrofisiológicos
- **Reentrada**: circuito com área de condução lenta + via rápida; precisa de **bloqueio unidirecional** e **períodos refratários desiguais**. CVE “zera” as fases e **interrompe o circuito**.
- **Atividade deflagrada** (*triggered*) — EAD/DAD: pós-despolarizações que atingem limiar; CVE pode suprimir **focos sincronizados** ao resetar o tecido.
- **Automaticidade anormal**: foco ectópico com limiar menor/ritmo mais rápido; CVE pode interromper, mas recorrência é comum se o **gatilho autonômico** persiste.

## 2. Por que sincronizar
- Pulso em **R** evita **R-on-T** (indução de TV/FV). Sincronismo é obrigatório para taquiarritmias com pulso.
- **Desfibrilação (não sincronizada)** é para **FV/TV sem pulso** e PCR.

## 3. Quem é cardioversável (com pulso)
- **FA/Flutter** com **instabilidade** (dor, hipotensão, hipoperfusão, edema agudo, alteração sensorial).
- **Taquicardias reentrantes** (AVNRT/AVRT) refratárias/instáveis.
- **TV monomórfica com pulso** **instável** (sincronizada). Polimórfica ou sem pulso → desfibrilar.

## 4. Por que a instabilidade manda
- Taquicardia rápida reduz **enchimento diastólico**, perde **sincronia AV** e cai **DC**; em **CMH**, isso é crítico.
- **Isquemia** surge por desequilíbrio oferta/demanda — daí a dor; **tempo** piora necrose e recorrência.
